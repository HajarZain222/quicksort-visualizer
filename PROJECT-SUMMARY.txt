DESIGN DECISIONS & OPTIMIZATIONS
================================================================================

1. SINGLE-FILE HTML VISUALIZER:
   • Eliminates deployment complexity
   • No build process needed
   • Easy to share and run
   • Self-contained and portable

2. WEB AUDIO API FOR SOUNDS:
   • Generates audio programmatically (no files needed)
   • Sine wave generation for start sound
   • Multiple oscillators for chime effect
   • Proper audio context management

3. CSS-BASED ANIMATIONS:
   • Hardware acceleration (GPU)
   • Smooth 60fps performance
   • Better than JavaScript animations
   • Staggered delays for visual effect

4. RESPONSIVE DESIGN:
   • Mobile-first approach
   • Flexbox for layout flexibility
   • Grid for statistics display
   • Touch-friendly interaction sizes

5. MEDIAN-OF-THREE PIVOT:
   • Reduces O(n²) worst case probability
   • Performs better on sorted/partially sorted data
   • Minimal overhead compared to random pivot
   • Industry-standard approach

6. HYBRID INSERTION SORT:
   • Faster than QuickSort for arrays < 10 elements
   • Automatic threshold switching
   • Significant performance improvement on small subarrays

7. THREE-WAY PARTITION:
   • Excellent for duplicate-heavy data
   • Reduces to O(n) for all-identical arrays
   • Significant speedup (2.5x-12000x in tests)
